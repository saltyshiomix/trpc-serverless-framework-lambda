# aws-lambda-trpc-serverless-framework-v3

![](./docs/images/hero.png)

## Setup

```
pnpm install
npx lefthook install
```

## Development

Lambda (Serverless Offline):

```
cd apps/api
pnpm local
```

Web (Next.js):

```
cd apps/web
pnpm dev
```

Then, see `http://localhost:3000`.
